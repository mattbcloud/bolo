How It Works

  1. Create a new game or join an existing one with a game ID
  2. Share the game ID with other players
  3. Wait for players to join (game ID must be the same)
  4. Once in the emulator, launch Bolo from the Infinite HD
  5. Set up or join a network game in Bolo

Note: Currently using single-instance mode. For multiplayer, players need to manually configure AppleTalk networking within the Mac emulator, or enable subdomain networking in app.js (requires DNS setup).

Quick Guide

Starting Bolo:

  1. Wait for Mac to boot (Happy Mac icon)
  2. Double-click "Infinite HD" on desktop
  3. Find and launch "Bolo"
  4. Choose "Join Network Game" or "Start Server"

Controls:

  • Click inside to capture mouse
  • Use Mac keyboard shortcuts
  • ESC to release mouse (if supported)
